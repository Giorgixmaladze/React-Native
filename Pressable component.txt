<Pressable> კომპონენტი, ისევე როგორც <Button>, საშუალებას გვაძლევს გავუშვათ კოდი მასზე დაჭერისას onPress prop-ის (თვისების) მეშვეობით. ამასთან, ის ასევე გვთავაზობს დამატებით წვდომის წერტილებს, როგორიცაა onPressIn და onPressOut, დაჭერის ქმედების უფრო დეტალური კონტროლისთვის.




<Pressable
  onPressIn={startAnimation}
  onPress={storeDraft}
  onPressOut={stopAnimation}
  { /* სხვა prop-ები */ }
>
  <Text>Store</Text>
</Pressable>


ამ მაგალითში ჩვენ ვიყენებთ:

    -onPressIn-ს startAnimation()-ის გამოსაძახებლად, რათა ეკრანზე ანიმაცია დაიწყოს.

    -onPress-ს მთავარი ლოგიკის, storeDraft()-ის გასააქტიურებლად.

    -onPressOut-ს stopAnimation()-ის გამოსაძახებლად, რომელიც ეკრანზე ანიმაციას აჩერებს.




გაითვალისწინეთ, რომ შესრულების თანმიმდევრობა მნიშვნელოვანია და შეგვიძლია წარმოვიდგინოთ ის, როგორც დაჭერის "სასიცოცხლო ციკლი" (lifecycle), რომელიც ასე მუშაობს:

onPressIn → onPress → onPressOut



კოდის მაგალითში შესაძლოა შენიშნეთ, რომ <Pressable> არ იყენებს title prop-ს. სამაგიეროდ, მისი კონტენტის საჩვენებლად ჩვენ გადავცემთ შვილობილ (child) კომპონენტს, ძირითადად <Text> კომპონენტს. ჩვენ ასევე შეგვიძლია შვილობილის სახით გადავცეთ ფუნქცია, რათა გამოვიყენოთ მისი დაჭერის მდგომარეობა (pressed state) შემდეგნაირად:


<Pressable
  { /* prop-ები */ }
>
  {(isPressed) => (<Text>{isPressed ? "Storing..." : "Store"}</Text>)}
</Pressable>




<Pressable>-ს აქვს სხვა ხშირად გამოყენებადი prop-ებიც, როგორიცაა:

    -disabled: თიშავს დაჭერის ქმედებას.

    -style: სტილის ობიექტი ან ფუნქცია (რომელიც იღებს მდგომარეობის ობიექტს), რომელიც აბრუნებს სტილის ობიექტს.

    -pressRetentionOffset: ცვლის მის გარშემო არსებულ დამატებით არეალს დაჭერის შესანარჩუნებლად.

    -hitSlop: ცვლის მის გარშემო არსებულ დამატებით არეალს დაჭერის გასააქტიურებლად.
