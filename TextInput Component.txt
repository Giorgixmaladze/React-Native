<TextInput> კომპონენტი მომხმარებელს ტექსტის შეყვანის საშუალებას აძლევს, რაც HTML-ის input-ს ან textarea-ს ჰგავს. ის აფიქსირებს მომხმარებლის მიერ შეყვანილ მონაცემებს და შეუძლია განაახლოს მდგომარეობა (state) წერის პარალელურად:


import { TextInput, View } from 'react-native';
import React, { useState } from 'react';

export default function TextInputExample() {
  const [text, setText] = useState('');

  return (
    <View>
      <TextInput
        style={{ height: 40, borderColor: 'gray', borderWidth: 1 }}
        value={text}
        onChangeText={setText}
        placeholder="ჩაწერეთ აქ..."
      />
    </View>
  );
}


porps:

  - value=> ის განსაზღვრავს, რა ტექსტი წერია ინპუტში მოცემულ მომენტში. ჩვეულებრივ, აქ ვწერთ ჩვენს state-ს.
  
  - onChangeText => ფუნქცია, რომელიც ეშვება ყოველ ჯერზე, როცა მომხმარებელი კლავიატურაზე ღილაკს აჭერს. ის ავტომატურად გვაძლევს ახალ ტექსტს, რომლითაც state-ს ვაახლებთ.

      შენიშვნა: ვებისგან განსხვავებით, აქ პირდაპირ ტექსტს ვიღებთ და არა event.target.value-ს.

  -placeholder: ეს არის ნაცრისფერი მინიშნება, რომელიც ჩანს მანამ, სანამ მომხმარებელი წერას დაიწყებს (მაგალითად: "ჩაწერეთ სახელი...").


  -maxLength: ზღუდავს სიმბოლოების რაოდენობას. თუ მიუთითებ 10-ს, მე-11 ასოს უბრალოდ აღარ დააწერინებს.


  -onFocus: ეშვება იმ წამსვე, როცა მომხმარებელი ინპუტში „შედის“

  -onBlur: ეშვება მაშინ, როცა მომხმარებელი ინპუტიდან „გამოდის“


  -editable: იღებს true ან false-ს. თუ false-ია, მომხმარებელი ინპუტს დაინახავს, მაგრამ შიგნით ვერაფერს ჩაწერს (იქნება Read-only).

  -keynoardType => არის ერთ-ერთი ყველაზე მნიშვნელოვანი prop მომხმარებლის გამოცდილების (UX) გასაუმჯობესებლად. ის განსაზღვრავს, თუ როგორი კლავიატურა ამოხტება ეკრანზე, როდესაც მომხმარებელი TextInput-ზე დააჭერს.

  წარმოიდგინე, რომ მომხმარებელს სთხოვ ტელეფონის ნომრის ჩაწერას, მაგრამ ეკრანზე ჩვეულებრივი, ტექსტური კლავიატურა გამოდის. მას მოუწევს სიმბოლოებში გადართვა, ციფრების ძებნა... ეს ზედმეტი წვალებაა. keyboardType-ით შენ ამ პრობლემას აგვარებ.


  ძირითადი ტიპები:

    -default => სტანდარტული კლავიატურა.

    -numeric => მხოლოდ ციფრების კლავიატურა.

    -email-address => ამატებს @ და . სიმბოლოებს მთავარ გვერდზე.

    -phone-pad => ტელეფონის ნომრის ასაკრეფი კლავიატურა.

    -decimal-pad => ციფრები და წერტილი/მძიმე.


    keyboardType="url" არის სპეციალური ტიპის კლავიატურა, რომელიც მორგებულია ვებსაიტის მისამართების (URL) სწრაფად და კომფორტულად ჩასაწერად.







  TextInput-ში ხშირად ვიყენებთ secureTextEntry={true}-საც. ეს პაროლის ველისთვისაა — ტექსტს წერტილებად აქცევს, რომ სხვამ ვერ დაინახოს.
