React Native-ში <Text> კომპონენტი არის ერთადერთი სწორი გზა ეკრანზე ნებისმიერი სახის ტექსტური ინფორმაციის გამოსატანად. ვებ-დეველოპმენტისგან განსხვავებით, სადაც ტექსტი შეგიძლიათ ჩაწეროთ <div>, <h1> ან <p> ელემენტებში, React Native-ში ტექსტის მოთავსება <View>-ში პირდაპირ შეცდომას გამოიწვევს.

---------------------------------------------------------------------
1. ძირითადი დანიშნულება
<Text> კომპონენტი გამოიყენება სტრიქონების (Strings) გამოსასახად. ის მხარს უჭერს ჩაშენებას (nesting), სტილიზაციას და მომხმარებლის ინტერაქციას (მაგალითად, დაჭერას).


2. ტექსტის ჩაშენება (Nesting)
React Native-ში შეგიძლიათ ერთი <Text> კომპონენტი მოათავსოთ მეორეში. ეს ძალიან გამოსადეგია, როდესაც გსურთ წინადადების მხოლოდ ერთ სიტყვას შეუცვალოთ ფერი ან სისქე (bold).

მაგ:
<Text style={{ fontSize: 18 }}>
  ეს არის ჩვეულებრივი ტექსტი, ხოლო 
  <Text style={{ fontWeight: 'bold', color: 'red' }}> ეს არის მუქი და წითელი.</Text>
</Text>

---------------------------------------------------------------

3. მნიშვნელოვანი თვისებები (Props):

    -numberOfLines: ზღუდავს ტექსტის ხაზების რაოდენობას. მაგალითად, თუ დაწერთ numberOfLines={2}, ზედმეტი ტექსტი აღარ გამოჩნდება.

    -ellipsizeMode: განსაზღვრავს, სად გამოჩნდეს მრავალწერტილი (...), როცა ტექსტი არ ეტევა (მაგ. head, middle, tail).

    -onPress: დიახ, ტექსტსაც შეიძლება ჰქონდეს დაჭერის ფუნქცია, რაც მას ბმულის მსგავსს ხდის.

    -selectable: განსაზღვრავს, შეძლებს თუ არა მომხმარებელი ტექსტის მონიშვნას და კოპირებას (ნაგულისხმევად false-ია).

---------------------------------------------------------------

4. სტილიზაციის თავისებურებები:

    -მემკვიდრეობითობა: ვებისგან განსხვავებით, სტილები არ გადაეცემა <View>-დან <Text>-ს. თუმცა, თუ გაქვთ ჩადგმული <Text> კომპონენტები, შიდა კომპონენტი დაიმკვიდრებს გარე კომპონენტის სტილს.


    -Layout: ტექსტს აქვს საკუთარი სპეციფიკური სტილები, როგორიცაა lineHeight, letterSpacing, textAlign და textDecorationLine.
