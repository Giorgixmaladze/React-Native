React Native არის პოპულარული Framework,რომელიც კომპანია meta მ 2015 წელს შექმნა. 


მისი მთავარი დანიშნულებაა მობილური აპლიკაციების შექმნა ios და android პლატფორმებისთვის ერთდროულად, მხოლოდ ერთი პროგრამირების ენის - javascript ის გამოყენებით.


მარტივად რომ ვთქვათ: ნაცვლად იმისა, რომ ცალკე წერო კოდი iPhone-ისთვის (Swift-ში) და ცალკე Android-ისთვის (Java/Kotlin-ში), React Native-ით წერ ერთხელ და ის მუშაობს ორივეზე.
